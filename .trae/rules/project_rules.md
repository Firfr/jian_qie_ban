# 项目规则  

## 命名规范

- 函数名称：以 `fn` 开头的中文
- 变量名称：以 `_` 开头的中文
- HTML 元素 ID：以 `id_` 开头的中文
- CSS Class：直接使用中文

## 环境与运行  
- 项目使用 bun 环境，禁止使用 Node.js 执行任何命令或依赖 Node 相关工具链。
- 使用 bun run dev 运行项目。不允许使用其他命令。
- `bun run dev` 已配置热重载，修改代码后无需手动停止或重启服务，AI 应默认此行为并避免提示“重启服务”等无关建议。  

## 端口处理  
若启动时报错端口占用，不要自动修改端口。  
应提示“端口占用”，并等待手动处理。
